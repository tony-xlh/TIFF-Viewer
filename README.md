# TIFF-Viewer
Demo of viewing TIFF files in the browser with Dynamic Web TWAIN

Demo links:

1. [index.html](https://tony-xlh.github.io/TIFF-Viewer/): display selected TIFF in a customized viewer.
2. [built-in-viewer.html](https://tony-xlh.github.io/TIFF-Viewer/built-in-viewer.html): display selected TIFF in a Dynamic Web TWAIN's built-in viewer.
